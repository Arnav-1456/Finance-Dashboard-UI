# Obsidian Flux - Finance Dashboard

> A premium, dark-mode-first financial analytics dashboard featuring reactive state management, intelligent insight computations, and strict role-based access constraints.

[![Live Demo](https://img.shields.io/badge/Live_Demo-Deploying_Soon...-brightgreen?style=for-the-badge)](#) *(Link will be added here!)*

## 🌟 Key Features

*   ** Obsidian Flux Design System:** Customized `@theme` directives leveraging soft neon glows, glassy backdrops, and interactive liftoff shadows avoiding standard borders.
*   ** Fully Responsive App Layout:** Collapsible sidebar mapping, modal-level navigations, and global search layouts natively built mobile-first.
*   ** Role-Based Access Control (RBAC):** Native toggle switching dynamically locking down routing constraints. "Admins" can edit and mutate the global ledger; "Viewers" are strictly locked to read-only mode.
*   ** Insight Engine:** Calculates net savings rates against a 20% margin algorithm, auto-detects maximum expenditures, and charts M/M comparative data.
*   ** LocalStorage Data Persistence:** Every transaction is cached synchronously in the browser layer. Zero setups required.
*   ** Comprehensive Test Suite:** Fully backed by `Vitest` & `React Testing Library` mapped against `happy-dom`.

## 🛠️ Tech Stack

*   **Framework:** React 19 + Vite 6
*   **Styling:** Tailwind CSS v4
*   **Charts:** Recharts
*   **Routing:** React Router v6
*   **Icons:** Lucide-React
*   **Testing:** Vitest / Happy-DOM / Jest-DOM

## 🚀 Getting Started Locally

Getting your own instance up and running locally is incredibly simple.

1. **Clone the repository:**
   ```bash
   git clone <your-repo-link>
   cd "Finance Dashboard UI"
   ```

2. **Install the dependencies:**
   ```bash
   npm install
   ```

3. **Start the local Vite server:**
   ```bash
   npm run dev
   ```
   Navigate to `http://localhost:5173/` in your browser.
